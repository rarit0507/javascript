자바스크립트의 출력 방향과 메시지창

스크립트의 적용 방식

  • 인라인 스크립트 : 해당 태그에 직접 스크립트를 기입하는 방식
  • 내부 스크립트 : 해당 문서에 script 태그를 기입하여 스크립팅하는 방식
  • 외부 스크립트 : 별도의 .js 파일을 작성하여 여러 문서에서 script 태그로 삽입하는 방식

스크립트의 출력 방향

  • 문서에 출력 : document.write()
  • 요소에 출력 : document.getElementById().innerHTML
  • 콘솔에 출력 : console.log()

메시지창

  • 알림창 : alert("메시지")
  • 확인창 : confirm("메시지")
  • 입력창 : prompt("메시지")
